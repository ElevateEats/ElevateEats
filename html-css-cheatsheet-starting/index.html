CSS Selectors and Specificity

Name Smybol Definition & Ex
ID Selector # Select single element and give it its own unique style. 1 value and 1 use per page. EX: #this-id
Type Selector N/A Matches type of element in html document. EX: "p" for <p> elements
Class Selector "." Match selector to class value you want to style, add "." in front. EX: .header for class="header" elements
Universal Selectors * Used to reset default browser styling or selecting all children of a parent.
Attribute Selector "[attribute]" we can use the attributes already attached to elements
Psuedo-Class Selector ":" This allows for changing the appearance of certain elements after user interaction. EX: :hover
